Delorean: Interpolation Preview

This extension shows an interpolation preview of the current glyph using two open fonts.

A Window to the Future

The preview window will update as you adjust contours while drawing so you can avoid surpises down the road. It's not meant for final interpolation, but works like a little window into your future interpoaltion process.

Initial Values

Delorean interpolates between Font [1] (Interpolation 0%) and Font [2] (Interpolation 100%). Font[1] is your CurrentFont() when Delorean is launched and Font [2] is the one right behind it. The initial interpolation value is set to 50. Delorean will display your CurrentGlyph() or you can type a glyph name into the "Glyph Name" box. The glyph must of course exist in both fonts, and the glyphs in both fonts must be compatible for interpolation.

Compatiblity

The glyphs must be compatible for interpolation to work with Delorean, as with any other interpolation process. If the glyphs aren't compatible, I would recommend using Prepolator. For final interpolation and for previewing multi-axis interpolation I would recommend using a more robust application like Superpolator which has many more advanced features.
